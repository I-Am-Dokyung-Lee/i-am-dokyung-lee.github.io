## 프로젝트 소개

취미와 여가활동을 탐색하고, 다양한 **원데이 클래스를 개설 및 신청**할 수 있는 웹 플랫폼입니다.
클래스 개설자(호스트)는 새로운 클래스를 등록하고 신청자를 모집할 수 있으며, 참여자(게스트)는 다양한 취미 클래스를 검색하고 간편하게 신청할 수 있습니다.

---

## 담당 역할

Spring Boot 기반의 백엔드 및 프론트엔드 개발에 참여했습니다.
AI 개발자로 전환하기 전, **웹 개발 전반의 기초를 다진 프로젝트**입니다.

---

## 주요 기능

| 기능 | 설명 |
|:---|:---|
| 소셜 로그인 | 카카오, 네이버, 구글 OAuth2 로그인 |
| 클래스 검색 | 제목/지역/카테고리/요일/난이도/금액 등 다중 조건 필터링 |
| 예약 및 결제 | 일정 선택, 쿠폰/크레딧 적용 결제 |
| 클래스 관리 | 호스트의 클래스 등록/수정/삭제 (CRUD) |
| 마이페이지 | 게스트/호스트 모드 전환, 위시리스트, 신청 내역 관리 |

## 포트폴리오에서의 의의

이 프로젝트는 AI 개발 이전에 **웹 서비스 개발 전체 사이클**(기획 → 설계 → 구현 → 배포)을 경험한 프로젝트입니다.
소셜 로그인, 다중 조건 검색, 결제 시스템 등 실제 서비스 수준의 기능을 구현하며 백엔드-프론트엔드 전 과정을 학습했습니다.
이때의 웹 개발 경험이 이후 Bizi 프로젝트에서 FastAPI 백엔드와 React 프론트엔드를 빠르게 이해하고 협업하는 데 기반이 되었습니다.
